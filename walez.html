WAL-EZ

A free Python script that uses the Selenium library to automate the Walmart purchase process

In Action!

Preface It'll blow your mind.

The story behind this little side project started at my grandma's house. I visited her very often as a child and when I was there also I used to play on this old Nintendo SNES with my family. It turns out the reason the SNES was there was because my older cousin used to play on it very much so as a child. Fast forward to 2017, Nintendo released the Super NES Classic Edition and my cousin asked me for help in acquiring this item, since it sells out extremely quickly. So to aid in the purchasing process, I created the initial version of this program to try and buy the SNES the night before one of Walmart's rare 11:00am sales. Unfortunately my first version failed to purchase the item fast enough due to some parts of the automated process failing. But now the version I have before you if used correctly is basically 100% reliable! So hopefully it'll help in whatever pickle you are in. :)

Installation See for yourself.

Make sure you have Python 3.6 installed! Once Python is installed make sure to have the WebDriver for Chrome installed as well so that Selenium can interact with Chrome! For Windows just unzip the chromedriver_win32 .zip file that you downloaded and make sure to put the chromedriver.exe found inside in the same folder as where you installed Python. Also make sure you have uBlock Origin as an extension for Chrome.

And lastly, this one. Checkmate.

To run the script right click the .py file and edit it with IDLE then press F5. Make sure all you know your Walmart login, and that the proper info ex. credit card and shipping address have been saved to your Walmart account. I reccomend not using your current Chrome user data(leaving the prompt blank or typing 'n'), and instead automatically sign in everytime you run the script, as using your current Chrome user data has been kinda spotty in my experience.

Good Luck and Enjoy!
